## Inheritance

Child class inherits attributes and methods from parent class.

```python
class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        return "Some sound"

class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name)  # Call parent constructor
        self.breed = breed
    
    def speak(self):  # Override parent method
        return "Woof!"

dog = Dog("Max", "Labrador")
print(dog.name)    # Max (inherited)
print(dog.breed)   # Labrador (own attribute)
print(dog.speak()) # Woof! (overridden)
